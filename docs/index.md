## 🎯 关于本站

本博客致力于为 **Java 开发者** 和 **AI 工程师** 提供系统化的学习路径和实用的技术资源。无论你是初学者还是经验丰富的开发者，都能在这里找到有价值的内容。

### 💡 为什么选择这里？

- ✅ **系统化学习路线** - 从入门到精通的完整路径
- ✅ **实战导向** - 注重实践，每个知识点都有应用场景
- ✅ **持续更新** - 紧跟技术发展，及时更新内容
- ✅ **清晰的结构** - 分阶段学习，循序渐进

### 📚 主要内容

#### Java 工程师技能树

涵盖从基础到进阶的完整技能体系：

- **基础阶段** - Java 语法、OOP、集合框架、多线程
- **进阶阶段** - JVM 深入、并发编程、性能优化
- **框架阶段** - Spring Boot 3.x、Spring Cloud、MyBatis
- **架构阶段** - 微服务、分布式系统、高并发解决方案

#### Python 与 AI 工程师技能树

从 Python 基础到 AI 应用的系统化路径：

- **Python 基础** - 语法、面向对象、标准库
- **Web 开发** - Flask 框架、RESTful API、数据库操作
- **数据处理** - NumPy、Pandas、数据可视化
- **机器学习** - 算法原理、Scikit-learn、模型评估
- **深度学习** - PyTorch、CNN、RNN、Transformer
- **专业方向** - CV、NLP、OCR、大语言模型

#### Linux 与 DevOps

现代开发者必备的运维技能：

- **Linux 基础** - 系统管理、网络配置、Shell 脚本
- **容器化** - Docker、容器编排
- **CI/CD** - GitHub Actions、Jenkins、自动化部署
- **监控运维** - 日志管理、性能监控、故障排查
