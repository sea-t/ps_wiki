## 🛠️ 技术栈

本站采用现代化的技术栈构建：

- **框架**: [VitePress](https://vitepress.dev/) - Vue 驱动的静态站点生成器
- **部署**: [GitHub Pages](https://pages.github.com/) - 免费的静态网站托管
- **CI/CD**: [GitHub Actions](https://github.com/features/actions) - 自动化构建和部署
- **搜索**: 内置全文搜索 - 快速查找内容

### 为什么选择 VitePress？

- ⚡ **极速构建** - 基于 Vite，构建速度极快
- 🎨 **现代 UI** - 简洁美观的界面设计
- 🔍 **强大搜索** - 内置全文搜索功能
- 🌓 **暗黑模式** - 保护眼睛的深色主题
- 📱 **响应式** - 完美支持移动端
- 🎯 **SEO 友好** - 服务端渲染，利于搜索引擎
